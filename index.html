Sobre a Universidade

O Estudo no HTML neste módulo: Começamos Aprendendo sobre as tags de cabeçalhos, as importantes tags <meta>, o significado de cada TAG, sobre tags de quebra de linha como <br> e de realizar uma linha com a TAG <hr>, por fim aprendido sobre as TAGS de cabeçalhos da página que vão do <h1> ao <h6>, indicando o grau de importância do título dentro de sua página HTML, não foi utilizado CSS em folha de estilo separada, por enquanto apenas começando o aprendizado, feito dentro de tabelas, mostrando como eram realizados o desenvolvimento de antigas páginas HTML.

Aprendido sobre as TAGS de parágrafo <p>, que são formatos block, temos também as TAGS de formatações de textos, como: <b>Negrito</b> ou de forma semântica <strong>Negrito</strong>.
<i>Itálico</i> ou de forma semântica <em>Itálico</em>.
<u>Underline</u>.
<strike>Texto Riscado</strike>.

Aprendido sobre listas ordenadas e não ordenadas em HTML, utilizando as TAGS <ul>Lista não Ordenada</ul>
<ol>Lista Ordenada</ol>
<li>Item da lista</li>

Por último foi aprendido também sobre as TAGS de imagem, links, tabelas e inputs.
<img>Imagens</img>
<a>Links como âncora</a>
<table>Tabelas</table>
